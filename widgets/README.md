# LifeSet Public Widgets

Embeddable web widgets for organisations to display on their websites.

## Available Widgets

### Schedule Widget
Displays class schedule for an organisation.

**Usage:**
```html
<script src="https://widgets.lifeset.app/schedule.js?orgId=org123"></script>
<div id="lifeset-schedule"></div>
```

### Memberships Widget
Displays available memberships for purchase.

**Usage:**
```html
<script src="https://widgets.lifeset.app/memberships.js?orgId=org123"></script>
<div id="lifeset-memberships"></div>
```

### Packs Widget
Displays available packs for purchase.

**Usage:**
```html
<script src="https://widgets.lifeset.app/packs.js?orgId=org123"></script>
<div id="lifeset-packs"></div>
```

## Hosting

Widgets can be hosted on:
- Firebase Hosting
- Vercel
- Any static hosting service

## Customization

Widgets respect organisation `brandColours` for theming.
